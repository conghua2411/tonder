# tonder

A tinder clone flutter app

## Getting Started

This is tinder clone app

Build with MVVM structure base on Android recommend app structure

using:
- provider (dependency injection)
- rxdart (data binding UI)
- sqlite (cache data)
- firestore (api network)
- mockito (unit test)
